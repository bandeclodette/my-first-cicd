# my-first-cicd   # CI/CD Pipeline Project
   
   ## Live Demo
   https://bandeclodette.github.io/my-first-cicd/
   
   ## Skills Used
   - GitHub Actions for CI/CD
   - YAML for pipeline config
   - Automated testing with grep
   - GitHub Pages deployment
   
   ## How it works
   1. Push code → GitHub Actions runs tests
   2. If tests pass → Auto-deploy to GitHub Pages
   3. Failed tests block deployment
